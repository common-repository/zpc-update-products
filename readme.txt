== Description ==
**Free** test version.  Let's you try the plugin before purchase. 
ZPC Update Products let you bulk edit your products. Raise or lower the sell price by a percentage or value. 
You can select products on Category or select several single products.  Prices can be changed manual, or in batch using a fied value or percentage.
The changes are previewed, green where price goes up, red where price goes down.


FEATURES:

Select products on categories, or choose a number of products.
Change price in bulk based on percentage or fixed value.
Preview changes. Higher price green, lower price in red.
Update products before or after applying bulk change.
Round price on chosen decimals (different rounding options)
Round to factor (eg prices always rounded on 5 cents).
Change stock quantities (Inline Edit).
Change product title (Inline Edit).
Visit (edit) product page
